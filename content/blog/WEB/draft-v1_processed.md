## ⚠️ 安全提醒：别用公司邮箱！

在动手之前，**有一件事必须提前说清楚**：

> 🚨 **请使用你的个人邮箱来玩，不要用公司邮箱。**

原因很简单：

- 公司邮箱里有**敏感信息**（商业机密、客户数据、内部文件）
- 把公司邮箱接给 AI，意味着 AI 可以**读取和发送**这些内容
- 万一出了问题，这个锅你背不起

**个人 QQ 邮箱、Gmail 这种，随便折腾。但公司邮箱——别碰。**

这不是技术问题，是安全意识。等你对这套流程足够熟悉了，再根据你们公司的安全策略决定是否接入。

好了，安全说完了，开始搞～

## 第一步：给龙虾装技能

在 OpenClaw 里装 Himalaya，一条命令搞定：

```bash
openclaw skill install himalaya
```

装完验证一下：

```bash
openclaw skill list
```

看到 `himalaya` 出现在列表里就 OK 了。

> 💡 和给手机装 App 一模一样——装了 App，手机就有了新能力。装了 Himalaya，龙虾就会收发邮件了。



## 第二步：让龙虾能登上你的邮箱

技能装好了，但龙虾还不知道怎么登你的邮箱。你需要做两件事：

### 2.1 先去 QQ 邮箱拿一把"专用钥匙"

你平时登 QQ 邮箱用的是 QQ 密码，但龙虾不能直接用这个——出于安全考虑，QQ 邮箱提供了一个叫**授权码**的东西，相当于给龙虾配一把专用钥匙：

- 🔐 只能收发邮件，不能登录你的 QQ
- ♻️ 你随时可以吊销
- 🛡️ 改了 QQ 密码，授权码自动失效

**怎么拿？三步：**

1. 电脑浏览器登录 [mail.qq.com](https://mail.qq.com)
2. 点右上角**「设置」→「账户」**，往下翻找到 **IMAP/SMTP 服务**，点「开启」
3. 手机短信验证后，页面弹出一个 **16 位授权码**

⚠️ **这个授权码只显示一次！看到就立刻复制保存。**

### 2.2 告诉 Himalaya 怎么连你的邮箱

钥匙拿到了，创建配置文件：

**macOS**：`~/Library/Application Support/himalaya/config.toml`

```toml
[accounts.qq]
default = true
display-name = "你的名字"
email = "你的QQ号@qq.com"

# 收邮件
backend.type = "imap"
backend.host = "imap.qq.com"
backend.port = 993
backend.login = "你的QQ号@qq.com"

[accounts.qq.backend.encryption]
type = "tls"

[accounts.qq.backend.auth]
type = "password"
command = "printf '%s' '你的16位授权码'"

# 发邮件
[accounts.qq.message.send.backend]
type = "smtp"
host = "smtp.qq.com"
port = 465
login = "你的QQ号@qq.com"

[accounts.qq.message.send.backend.encryption]
type = "tls"

[accounts.qq.message.send.backend.auth]
type = "password"
command = "printf '%s' '你的16位授权码'"
```

核心就三件事：

1. **邮箱在哪** → `imap.qq.com` 和 `smtp.qq.com`（QQ 邮箱标准地址，照抄就行）
2. **用什么钥匙** → 刚才拿到的 16 位授权码
3. **你是谁** → 你的 QQ 邮箱地址



## ✅ 验证：试试看

配好之后，回到终端验证一下：

```bash
himalaya account list
```

看到你的 QQ 邮箱出现了，说明连上了：

```

| qq   | IMAP, SMTP | yes     |
```
