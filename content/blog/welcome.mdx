## 👋 你好，欢迎来到我的博客！

这是我的个人网站，在这里我会分享：

- **AI 技术实践**：大语言模型、RAG、Text2SQL 等实战经验
- **学习笔记**：计算机视觉、高性能计算、算法优化

## 为什么要写博客？

写作是最好的学习方式之一。通过将所学整理成文字，可以：

1. 加深对知识的理解
2. 建立个人知识库
3. 与社区交流分享

## 技术栈

- **框架**: Next.js 16 + React 19
- **样式**: Tailwind CSS 4
- **内容**: Contentlayer + MDX
- **部署**: 支持 GitHub Pages / 自托管
